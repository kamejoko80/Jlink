#### [RTT](RTT/) 

RTT（ Real Time Terminal）是SEGGER公司新出的可以在嵌入式应用中与用户进行交互的实时终端。J-Link驱动4.90之后的版本都支持RTT。

RTT既可以从MCU上输出信息、也可以向应用程序发送信息，由于其高速的特性，所以不影响MCU的实时性，实现原理就是J-link与MCU共享内存。
